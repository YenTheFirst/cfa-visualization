API reference:

  • /oauth/signup will start the oauth authentication process for a signed-in foursquare user
  • / shows this page
  • /all_checkins returns JSON with all authenticated users most recent checkins. it takes a 'before' and/or 'after' parameter to limit what entries are returned. 'before' and 'after' are in unix epochs
    example: /all_checkins?after=1280722062&before=1286929463
